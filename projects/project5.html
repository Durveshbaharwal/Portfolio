Nakshatra Navigators Space Tourism Website

Jorney through the Solar System and beyond!

Project Overview

Welcome to Nakshatra Navigators, a project focused on space exploration, celestial navigation, and 3D visualizations. I have Designed and developed a website for Space Tourism encoraging space exlporers to find out ways to visit the planets in our galaxy and beyond. Also, I have Integrated 3D models to enhance visualization.

Technologies: HTML, CSS, JavaScript, THREE.js, blender.
Duration: 1 week

Project Preview

Open Github Repository

Repository Structure

1. 3dmodels/: 3D models for celestial bodies.


2. Images/: Relevant images for visualization.


3. videos/: Video resources on celestial navigation.


4. script.js: JavaScript for interactivity.


5. index.html: Main web interface.


6. style.css: Stylesheet for the interface.


How to Use This Repository

Simply open the index.html file in any browser to interact with the project.


Clone and Use

Clone the repository:

To get started, clone this repository to your local machine using the following command:

git clone https://github.com/Durveshbaharwal/Nakshatra-Navigators.git
cd Nakshatra-Navigators

Copied to clipboard!


Features

3D Models: Visual representations of celestial bodies and constellations.


Web-based Interface: A clean UI powered by HTML, CSS, and JavaScript.


Interactive Features: Explore videos and images related to the Nakshatra system and celestial navigation.
